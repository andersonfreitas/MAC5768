Exerccio 1

A imagem abaixo foi gerada com a funo iacos da toolbox ia636. Leia a documentao da funo e explique a imagem gerada pelo comando iacos(s,t,theta,0)

In [6]: 
from ia636 import *

s = (256,256)
t = 64
theta = pi/2
f = iacos(s, t, theta, 0)
imshow(f)
gray()

Resposta 1

O mtodo iacos() cria uma imagem em tons de cinza com uma representao cosenoidal com amplitude $1$ na dimenso s, com o perodo t. A orientao da imagem ajustada com o parmetro $\theta$ e offset com o parmetro $\phi$

Internamente a imagem gerada atravs da equao abaixo para cada $x$ e $y$:

$$ I_{ x,y }=\cos ({ 2\pi [(\frac { 1 }{ t } \cos { \theta } )x+(\frac { 1 }{ t } \sin { \theta } )y]+\phi }) $$
